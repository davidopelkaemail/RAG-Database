🤔 Why do we need to slice documents?

When AI reads a huge document, it's like trying to eat an entire pizza in one bite! We need to slice it into smaller pieces so AI can understand each part better.

Real Example: If you have a 100-page book, AI might read it 10 pages at a time instead of all at once!

Vector Detective! 🔍

Use the magic magnifying glass to find similar ideas! When you search for something, AI looks for similar VectorsSpecial number codes that help AI find matching ideas in its memory.

🔍

Click the magnifying glass to search for similar ideas!

Try searching for "animal" to see similar ideas!

🕵️ How Vector Detective Works:

When you type something, AI converts it into a special number code (vector). Then it searches its memory for similar codes! It's like having a super-fast librarian who finds books on similar topics instantly.

Example: If you search for "animal", AI might find "dog", "cat", "pet" - all the related ideas!

⚙️ How the RAG Pipeline Works:

1. Documents - AI starts by reading information
2. Chunks - Breaks big documents into smaller pieces
3. Vectors - Creates magic number codes for each piece
4. Database - Stores all the magic codes
5. Query - AI receives a question to answer
6. Answer - AI uses the stored information to give the best answer!
